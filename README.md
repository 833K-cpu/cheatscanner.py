# Cheat Scanner

### Download and How To Use

1. [Download the latest cheatscanner.exe here](https://github.com/833K-cpu/cheat-scanner-release/releases/latest)
2. Run cheatscanner.exe from the download location
3. Use it like this:

```
cheatscanner.exe "C:\Path\To\Your\Mods\Folder"
```
Or drag a `.jar` or folder onto the executable (if your Windows setup allows).
